# Quick Start Guide

## Prerequisites
- Node.js 18+
- Python 3.11+
- Docker Desktop

## Setup

1. Clone repository
2. Run setup: `./scripts/setup.sh`
3. Edit `.env` with API keys
4. Start: `make dev`

## API Keys Needed

- MOT API: https://documentation.history.mot.api.gov.uk/
- OpenAI: https://platform.openai.com/
- Stripe: https://stripe.com/
- Resend: https://resend.com/

## Access

- Frontend: http://localhost:3000
- Backend: http://localhost:8000
- API Docs: http://localhost:8000/docs

For detailed instructions, see SETUP_CHECKLIST.md
